2. Explain Local Scope, Block Scope, Functional Scope and Scope Chain in javascript.

  • Local scope or block scope in JS means the variable declared within {}, if, switch, loops then it can only be accessed inside that block and not outside it.
  • Functional scope is name as local scope, the only difference is that the variable is defined inside a function.
  • Scope chain means the variables of the lexical parent can be accessed inside a block/function.

3. What is difference between null and undefined and where to use what?

Null and undefined are primitive data types. Null is like a keyword that is programmatically empty and undefined is a keyword that will be getting if you are accessing variables before initialisation.

undefined is the default value before initialisation while null is a specified value as 'nothing' by the developer.

4. What is Symbol?

Symbol is primitive data type introduced in ES6 version of JS. They are immutable i.e which cannot be changed and unique.

it will return Boolean values.

        const val = Symbol(1);
        const val2 = Symbol(1);
        console.log(val == val2);

5. Write code to explain map and filter in arrays.

Map will iterate over all the elements on which it has been used, it will return a new array.

Filter will return a new array after checking a condition on each element of the old array.

        let a = [10,20.30,40]
        let c = a.filter(item=> item >10)
        console.log(c)
        let b = a.map(item => item *2);
        console.log(b)

6. Explain passed by value and passed by reference.

Passed by value works with primitive data types. It means that every time we assign a value to a variable a copy of that variable is created every single time.

passed by reference works with non-primitive data types. When you are creating an object you're given a reference to that object, if 2 variables hold the same reference, then changing the object reflects in both variable.

        
        let a = 10;
        let b = a;
        b = a + 5;
        console.log(a)
        console.log(b)

        
        let obj1 = {
            name: "bhanu",
            rollNo: 115;
        }
        let obj2 = obj1;

        obj2.rollNo = 116;
        console.log(obj1);
        console.log(obj2);
